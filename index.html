Directions

The purpose of this App is to generate a standard seed phrase using true and verifiable randomness. Currently most hardware wallets generate the randomness for you. Using this app you can create real randomness by flipping a coin, map it to a seed phrase, and then input it into the hardware device.

You will be flipping a coin(s) for the randomness. As you do, the 1s (heads) and 0s (tails) will be directly mappable to your seed words using this chart. For instance, the bitString '00101101001' produces the word 'coffee'.

So why use any software at all? Unfortunately the specification for Bip39 has a checksum appended to the end that requires sha256 to compute. This was a design mistake in my opinion. Simpler checksum methods exist that would have enabled doing this process on a paper worksheet.

This process is recommended due to it being verifiable. Input your bit string into the textbox. Once your words are displayed, use the paper chart (which you should print out) to make sure all your words match. The last few bits will not make a full word, and the checksum that gets generated determines the last few bits for the final word.

This software is online! thats a problem. When creating a cold wallet, you do not want to input the entropy to a Internet-connected computer. My advise is to buy an old iphone, ipod touch, or raspberry pi from a pawn shop, navigate to the page, then turn off the network connection (all of the calculations are done client side). Complete the process, write down the seed phrase, and then destroy the device.

Once you have your seed phrase, input it in your hardware wallet using the "restore" or "import" feature. This means you already have a key (you just made it). All hardware wallets have this feature.

Send in a small amount into the first 2 addresses.

Either input the phrase into another (different) hardware wallet, or reset the hardware wallet and again restore it from your seed phrase. Make sure it produces the same addresses. Use this second wallet to spend the small amount out of the 2 addresses. This verifies that your paper backup is working correctly.
